文字格式

Markdown 是当下流行文档书写语言，旨在通过简单的语法实现对常见格式的支持。

标准 Markdown 支持 粗体 和 斜体 文本，部分实现支持 删除线。下划线则需通过内嵌 HTML 实现，像是这样。

还有上标 X2 和下标 Y3，将文字高亮起来，属于扩展格式，手动支持了。
