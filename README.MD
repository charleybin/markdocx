# markdocx

将你的 markdown 文件转换为 MS Word（.docx）/ Convert your Markdown files to MS Word (.docx).

正在开发中 / Under development

## 测试阶段使用方法 

1. 需要 Python 2.7+ 环境
2. 安装依赖 `pip install -r requirements.txt`
3. 打开 `src/test/test_main.py`，根据注释修改里面的路径
4. 运行上述 `test_main.py`
